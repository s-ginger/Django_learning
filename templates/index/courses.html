О курсе

Этот курс научит тебя создавать настоящие веб-приложения с нуля. Ты изучишь HTML, CSS, JavaScript, React, Python, Django и многое другое. Подходит как новичкам, так и тем, кто хочет улучшить навыки.

Что входит в курс?

Модуль 1: Вёрстка

HTML, CSS, Bootstrap — создаём современный интерфейс.

Модуль 2: JavaScript & React

Фронтенд магия — интерактивность и компоненты.

Модуль 3: Django & Python

Бэкенд, базы данных, API и продакшен.
